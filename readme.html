RTJam - Real Time Jam

RTJam is software that runs on Linux (Raspberry Pi OS and others) that uses low latency Voice over IP (VoIP) to create a virtual room for musicians to collaborate in real time. It's just super low latency audio conferencing. Time delay from player to player is about 25msec. I needed to get my music fix during the pandemic and nothing else I could find worked reliably.

The code is open source and can be found at GitHub

You can listen in to the lobby of the jam room live! (see the calendar below for when something might be happening there)
